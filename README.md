## 我在做什么

我正在把 AI 工作流、文档工程和平台化产品揉在一起，最近的主线是 [Presto-io](https://github.com/Presto-io)：一个 Markdown -> Typst -> PDF 的文档转换平台。

Presto-io 不只是一个转换器。它包含桌面端、Web 服务、官网、模板注册中心、官方模板、脚手架、开发者文档和 Agent Skills。我的工作重点是把这些部分组织成一个能持续演进的生态：

- **主应用**：Go + SvelteKit + Wails，桌面端与 Web 端共享核心逻辑。
- **模板系统**：用独立二进制协议连接 Go、Rust、TypeScript 等不同语言的模板。
- **注册中心**：GitHub Actions + Python 生成静态 registry.json，通过 Cloudflare Pages/CDN 分发。
- **安全模型**：official、verified、community、unrecorded 四级信任体系，配合 SHA256 校验、下载白名单、超时和最小环境执行。
- **开发者体验**：维护 create-presto-template、多语言 starter、官方模板和完整模板开发规范。
- **文档与官网**：用 Astro 官网、真实 UI Showcase 和集中式文档仓库服务用户、模板开发者和核心贡献者。
